01 Editorial · magazine

Dispatch — the editorial

A magazine. Big serif display, dispatch numbers, masthead, pullquotes. Treats every product launch like a feature article and every collab like a guest byline. Reference: Tracksmith, Bandit Running editorial.

Type  Fraunces + Newsreader + IBM Plex Mono Open →
02 Technical · spec sheet

Field Manual — the technical

A sport-science manual. Mono-heavy, court diagrams, ISO-style classifications, tabular spec rows. Treats the kit as a parts list and Osaka as a published field test. Reference: Satisfy, Hoka tech pages.

Type  JetBrains Mono + Manrope Open →
03 Brutalist · poster

Signal — the loud one

Monumentally large display type, signal-orange tactical accent, marquee tickers, asymmetric blocks. Treats every drop like a streetwear release and the Osaka collab as a stamped capsule. Reference: Highsnobiety, early Nike SB.

Type  Archivo Black + Manrope + JetBrains Mono Open →
04 Wayfinding · map

Atlas — the cartographic

Houston padel rendered as a transit map. Coordinates, dotted-line routes, location pins, geographic dispatches. Treats Osaka as a satellite outpost on the map. Reference: vintage transit maps, expedition logs.

Type  DM Serif Display + DM Sans + DM Mono Open →
05 Lookbook · gallery

Exhibition — the gallery

A curated lookbook. Generous whitespace, light italic serif, plate numbers, very quiet. Treats the spring kit as an exhibition in five plates and Osaka as a guest residency. Reference: Aimé Leon Dore, Satisfy lookbooks.

Type  Cormorant Garamond + Manrope Open →
For the partnership conversation

Each direction includes a distinct Osaka treatment.

So your friends can see how a collab/partnership renders five different ways: as an editorial feature (Dispatch), a published field test (Field Manual), a stamped drop card (Signal), a satellite outpost on the map (Atlas), and a guest curator residency (Exhibition).

How to review
this deck.

01Click each tile  Each design is a fully working single-page concept — hero, products, schedule, Osaka section, manifesto, footer. Open them in separate tabs.
02Pay attention to typography first  Type is the strongest signal of a design's POV. Fraunces (Dispatch) feels nothing like Archivo Black (Signal) feels nothing like Cormorant (Exhibition).
03Then look at the Osaka section  Each one renders the partnership differently. That's the canary for whether you'd want this brand to host other collabs the same way.
04Forget which one looks "newest"  Pick the one your friends' actual customer would feel proud to wear out of the house. The kit is unisex — the design should be too.
05Tell us a winner  We'll bring the chosen direction up to a full storefront with real product detail pages, cart, and the rest of the page tree.
